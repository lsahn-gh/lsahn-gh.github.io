First and Follw Sets

대학교에서 First and Follow sets을 처음 배울때 follow는 이해하기가 어려웠습니다. 따라서 제가 배운 규칙들을 다시 작성하였으며 도움이 되길 바랍니다. 이 규칙들이 올바른지 궁금할 것 입니다. 그래서 저는 한 강사님에게 수업에서 이 방법을 시도해 볼 것을 요청하였고 효과가 있다고 생각합니다. 궁금한 점이 있거나 무엇인가 제안하고 싶다면 자유롭게 메일1로 연락주시기 바랍니다.

Rules for First Sets

  1. X가 terminal 이라면 First(X)는 X입니다.
  2. Production X → ε이 존재하면 ε을 first(X)에 추가하세요.
  3. Production X → Y1Y2..Yk가 존재하면 first(Y1Y2..Yk)를 first(X)에 추가하세요.
  4. First(Y1Y2..Yk)가 다음과 같다면..
    1. First(Y1) (First(Y1)에 ε이 포함되지 않은 경우)
    2. 또는 (First(Y1)에 ε이 포함)이 된다면 ε을 제외한 모든 First(Y1Y2..Yk)는 First(Y1)에 포함됩니다. 이는 First(Y2..Yk)에도 적용됩니다.
    3. ε이 First(Y1) First(Y2) .. First(Yk)에 포함이 되면 ε을 First(Y1Y2..Yk)에 추가하세요.

Rules for Follow Sets

  1. Follow(S)에 $을 추가하세요. (S는 시작 심볼입니다.)
  2. Production A → aBb가 존재하면 ε을 제외한 모든 FIRST(b)를 FOLLOW(B)에 추가합니다.
  3. Production A → aB가 존재하면 FOLLOW(A)에 존재하는 모든 것을 FOLLOW(B)에 추가합니다.
  4. production A → aBb가 존재하고 FIRST(b)에 ε이 포함되면 FOLLOW(A)의 모든 것을 FOLLOW(B)에 추가합니다.
