Python - Auto generate requirements.txt

Mon, May 18, 2015 python, requirements.txt

원문 – Python - Auto generate requirements.txt by Suresh Kumar
번역 – Yi Soo, (Thomas) An

Why requirements.txt?

모든 애플리케이션은 실행될 때 요구하는 종속된 라이브러리들을 가지고 있습니다.
requirements 파일은 한 번에 어떠한 라이브러리가 종속적인지 확인하고 설치하도록 도와줍니다.

requirements 파일 형식:

requests==1.2.0
Flask==0.10.1

방법 1:

$ pip freeze > requirements.txt

requirements.txt 파일을 생성하기 위해 pip의 freeze 명령어를 사용합니다.:
위의 포맷 형식으로 requirements.txt을 생성했다면, pip install -r requirements.txt로 사용할 수 있습니다.

방법 2:

pipreqs 사용하기 - pipreqs란 프로젝트에 사용된 패키지만 이용하여 requirements.txt 파일을 생성하는 방법입니다.

왜 pip freeze가 아닌가요?

  • virtualenv를 사용하지 않는 한 pip freeze는 프로젝트에서 사용하지 않는 패키지도 모두 포함합니다.
  • pip freeze는 오로지 당신의 개발 환경에 설치된 패키지들만 사용하여 구성합니다.
  • 때로는 패키지를 설치하지 않고 requirements.txt를 생성할 필요가 있습니다.

어떻게 사용하나요?

$ pip install pipreqs
$ pipreqs /path/to/project
