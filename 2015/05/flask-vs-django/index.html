Pirates use Flask, the Navy uses Django

Sun, May 3, 2015 flask, django, python

원문 – Pirates use Flask, the Navy uses Django by Alan Hamlett
번역 – Yi Soo, (Thomas) An

Introduce

새로운 아이디어를 테스트하거나, 제품을 개발하고자 한다면, 빌드하기 위해 Web stack을 선택해야 합니다. 파이썬 개발자를 위해 두 개의 유명한 웹 프레임워크 Flask와 Django가 있죠. I have experience with both and have chosen one or the other for my myriad of projects and copanies. My current product, WakaTime는 Flask로 개발 되었고, Flask는 우리가 목표에 도달할 수 있도록 도와주었습니다.

경험이 있다는 것은 올바른 프레임워크를 선택하도록 도와줍니다. 개발자가 결정하는데 도울 수 있도록 워크시트를 제작하였습니다. Django vs Flask worksheet에서 최적의 프레임워크를 선택할 수 있도록 확인해보세요.

Performance

Flask는 Django보다 좀 더 빠르게 JSON을 반환합니다.

하지만, 타 언어 프레임워크와 비교하자면 둘의 차이는 신경쓸 필요가 없습니다. Django와 Flask를 사용하는 이유는 개발 속도 향상, 빠른 빌드, 충분히 빠른 프레임워크이기 때문이죠.

마지막으로, Django와 Flask를 사용하는 이유는 개발시간 단축과 빠른 빌드입니다. 제품개발에 있어 어느 프레임워크가 최적인지 결정할 때, 이 워크시트와 당신의 결정을 고려하여 선택하세요.

또한, 저는 당신의 결정을 돕고 의견 나누길 희망합니다. 여기에 질문을 남기거나, irc.freenode.net의 #wakatime에 방문하세요.

P.S. Pirates use Flask, the Navy uses Django. WakeTime는 해적선입니다 ;)
