Python - Auto generate requirements.txt

Why requirements.txt? 모든 애플리케이션은 실행될 때 요구하는 종속된 라이브러리들을 가지고 있습니다. requirements 파일은 한 번에 어떠한 라이브러리가 종속적인지 확인하고 설치하도록 도와줍니다. requirements 파일 형식: requests==1.2.0 Flask==0.10.1 방법 1: $ pip freeze > requirements.txt requirements.txt 파일을 생성하기 위해 pip의 freeze 명령어를 사용합니다.: 위의 포맷 형식으로 requirements.txt을 생성했다면, pip install -r requirements.txt로 사용할 수 있습니다. 방법 2: pipreqs 사용하기 - pipreqs란 프로젝트에 사용된 패키지만 이용하여 requirements.txt 파일을 생성하는 방법입니다. 왜 pip freeze가 아닌가요?
