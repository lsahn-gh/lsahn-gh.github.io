Learning MSP430F5529 - # Interrupt

이 글은 인터럽트 예제를 분석한 글이다, 어느 누군가에게 도움이 되길… 아래 코드는 ti.com에서 제공하는 인터럽트 예제코드1이다. … #define RED_LED P1OUT // On-board RED led. int main(void) { P1DIR |= 0x01; // Set direction OUT the led. TA0CCTL0 = CCIE; TA0CCR0 = 50000; TA0CTL = TASSEL_2 + MC_1 + TACLR; __bis_SR_register(LPM0_bits + GIE); } … #pragma vector=TIMER0_A0_VECTOR __interrupt void TIMER0_A0_ISR(void) { RED_LED ^= 0x01; } GIE - General interrupt enable의 약자로 MCU의 인터럽트 허용여부를 결정한다.
